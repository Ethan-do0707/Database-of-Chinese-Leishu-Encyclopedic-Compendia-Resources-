# 仓库简介文案

## GitHub About 短版

《渊鉴类函》结构化缓存数据仓，适用于古典诗词用典检索、诗对联想、古籍语料构建与数字人文研究。

## 仓库简介标准版

本仓库收录《渊鉴类函》的结构化分页缓存数据，按 `JSON` 分片组织，包含部类信息、正文摘录与词目分组，可直接用于典故检索、诗对联想、古籍语料处理、搜索索引构建与数字人文分析。

## 仓库简介扩展版

这是一个面向古典诗词与古籍计算应用的《渊鉴类函》结构化数据仓。仓库采用一页一文件的 `JSON` 分片方式，保留部类路径、正文摘录与词目分组信息，适合用于典故检索、诗对推荐、语料清洗、知识库构建与数字人文研究场景。
