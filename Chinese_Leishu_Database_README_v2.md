# 📚 Chinese Leishu Database  
# 中国类书资源库

---

## 🌟 Overview | 项目简介

**Chinese Leishu Database** is a structured digital repository that collects and organizes materials from traditional Chinese *leishu* (encyclopedic compendia). It aims to provide researchers with accessible, computable, and reusable data resources for the study of classical Chinese texts.

**中国类书资源库**致力于收集和整理中国古代类书文献，将其转化为结构化数据，为研究者提供便捷、可计算、可复用的资源支持，服务于古典文献研究与数字人文领域。

---

## 🎯 Vision | 项目愿景

> From *Leishu* to Large Language Models: building a structured knowledge infrastructure for classical Chinese intertextuality.  

> 从类书到大模型：构建面向古典文献互文研究的结构化知识基础设施。

---

## 📖 What is Leishu? | 什么是“类书”？

*Leishu* are traditional Chinese encyclopedic compilations that categorize knowledge from earlier texts into thematic entries.

类书是中国古代的一种百科式文献，通过分类汇编前代典籍内容，形成以主题为核心的知识体系。

## 🔗 Relationship with Duoshi | 与 Duoshi 模型的关系

This repository serves as the **data infrastructure** for the Duoshi model.

- Leishu Database → provides structured knowledge (事)  
- Duoshi Model → performs semantic interpretation (意)  

Together they form a unified framework:

```
Meaning (意) ↔ Allusion (事) ↔ Text (文)
```

---

本项目构成 Duoshi 模型的**数据基础设施**：

- 类书数据库提供“事”的知识结构  
- Duoshi 模型完成“意”的语义解释  

共同构建：

```
意 ↔ 事 ↔ 文
```
